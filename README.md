
##### `delete_techsignal_by_name.py`
Скрипт удаляет технологические сигналы по ключевому слову в имени.
```
isim_ip = "10.50.94.12"         # IP адресс
isim_login =  "login"           # Логин
isim_pass = "password"          # Пароль
key_word = "area"               # Ключевое словое для поиска
delete = 0                      # 0 - Выведет на печать найденные сигналы
                                # 1 - Удалить найденные сигналы
```
###### Тестировалось на 5.5

##### `delete_techsignal_by_mac.py`
Скрипт удаляет технологические сигналы по мак адрессу.
```
isim_ip = "10.50.94.12"         # IP адресс
isim_login =  "login"           # Логин
isim_pass = "password"          # Пароль
mac = "FA:16:3E:F2:10:70"       # mac для поиска
delete = 0                      # 0 - Выведет на печать найденные сигналы
                                # 1 - Удалить найденные сигналы
```
###### Тестировалось на 5.8
